About This Project

This application was developed for the 2025 NASA Space Apps Challenge. It uses NASA Earth observation data to provide historical weather probability analysis for outdoor event planning.

The system leverages multiple NASA datasets including:

  • GLDAS Noah Land Surface Model data for temperature and soil moisture
  • MERRA-2 land surface data for additional climate variables
  • GPM IMERG precipitation data for rainfall analysis
  • VIIRS cloud mask data for cloud cover assessment

By analyzing decades of historical data, the application calculates the probability of adverse weather conditions for any location and date, helping users make informed decisions about their outdoor activities.
